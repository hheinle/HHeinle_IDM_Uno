Variante : regles inventées 1

Configuration de départ :

Pour cette variante, chaque joueur se voit distribuer 3 main(s) de 4 cartes. Une main de joueur doit être cachee.

Il doit y avoir 3 talon(s) où poser les cartes jouées.

Il doit y avoir une rivière. C'est-à-dire qu'il doit y avoir 4 cartes à côté du talon.

Les joueurs disposent de 9 secondes durant leur tour pour jouer leur(s) carte(s).

Déroulement d'une manche :

Un joueur peut poser plusieurs cartes par tour.

Inversion du sens de jeu :

Le sens du jeu peut changer à l'aide de :

  • la carte inversion classique
  • les cartes avec le chiffre sept

Tourner les mains :

Tous les joueurs échangent leurs mains dans le sens du sens actuel à l'aide de :

  • la carte UNO
  • Les cartes avec le chiffre zero

Pioche :

Quand un joueur doit piocher, il peut piocher jusqu'à trois cartes.

Après avoir piocher, le joueur a le droit de jouer.

Contestations :

Les adversaires peuvent contester l'action du joueur :

  • s'il fait une interception
  • s'il pose une carte +4.

Il est possible de cumuler les cartes :

  • les cartes +2
  • les cartes +4
  • les cartes couleur
  • les cartes Uno

Effets sur les cartes 1 à 9 :

  • les suites sont autorisées
  • les suites royales sont autorisées
  • il est possible d'interrompre ces suites en posant une carte de même nombre et de même couleur

Fin d'une manche :

Les joueurs doivent annoncer "UNO" en jouant leurs avant-dernières cartes.

Le premier joueur à s'être débarassé de toutes ses cartes remporte la manche. Ce joueur ne marque pas de points mais comptabilise une manche gagnée.

Les perdants ne marquent pas de points.

Fin de la partie :

Le vainqueur de la partie est le joueur qui comtabilise le plus de manche gagnées.
